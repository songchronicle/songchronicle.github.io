Диаграмма распределения песен по годам

Страница пытается автоматически загрузить файл songs.json из той же папки. Если браузер блокирует чтение локального файла, выбери JSON вручную через кнопку ниже.

Ожидание загрузки данных…
Ожидается JSON-массив объектов с полями artist, title, year. Для автоматической загрузки удобнее запускать файл через локальный сервер, например: python -m http.server
